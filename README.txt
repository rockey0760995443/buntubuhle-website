Buntubuhle Talent & Tech - Website Package
Files included:
- index.html
- styles.css
- script.js
- assets/logo.png (your uploaded logo)
- assets/Buntubuhle_services.pdf (if uploaded)
- assets/Welcome_message_and_contacts.pdf (if uploaded)

How to use:
1. Unzip the package.
2. Open index.html locally to preview the site.
3. To host: upload all files to GitHub Pages, Netlify, or your hosting provider.
   - For GitHub Pages: create a repo, push files to main branch, enable Pages in repo settings.
   - For Netlify: drag-and-drop the folder in the Netlify dashboard (Sites -> New site -> Deploy manually).
Form behavior:
- The contact form uses mailto: to open the user's email client and prefill the message to buntubuhleva@gmail.com.
- To add server-side form handling, replace the form action with your Formspree/EmailJS endpoint.
